This repo is a minimal, ready‑to‑use starter template for building static HTML projects with Tailwind CSS — ideal for simple prototypes, landing pages, or student projects that don’t require a complex build system. It includes HTML, CSS and Tailwind config files, and is set up so that when you run npm run build, Tailwind will compile only the classes you use into a final stylesheet (output.css), keeping the output clean and optimized.

You can view the project here: BP_HTML_10TWND

Installation & Deployment

Local Setup

npm install – installs all dependencies from package.json into node_modules.

npm run build – runs the build script to compile Tailwind CSS, process assets, and output a production-ready site (usually in dist/ or public/).

Netlify Deployment

Connect your GitHub repo; Netlify automatically runs npm install and npm run build on the server.

Set Build command to npm run build and Publish directory to your build folder (e.g., dist/).

Using Images

Place images in public/ and reference them from root or Tailwind: bg-[url('/img/img.jpg')]. Ensures images are copied to the build output and work both locally and on Netlify.
